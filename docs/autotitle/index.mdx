# Autotitle

**Automated media file renamer using online databases.**

Autotitle scans your anime/media directory, matches filenames against online databases (MyAnimeList, AnimeFillerList), and renames them into a clean, consistent format. It also detects and tags filler episodes automatically.

## What it does

```text
BEFORE:
[SubGrp] Attack on Titan - S01E01 [1080p].mkv
[SubGrp] Attack on Titan - S01E02.mkv

AFTER:
Attack on Titan - 01 - To You, in 2000 Years.mkv
Attack on Titan - 02 - That Day.mkv
```
